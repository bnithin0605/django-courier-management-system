# 📦 Courier Management System (Django)

A web-based Courier Management System built using Django to streamline shipment tracking, delivery workflow management, and administrative operations.

This system provides role-based access for administrators, staff, and customers, enabling efficient parcel tracking, status updates, and delivery management.

---

## 🚀 Project Overview

The Courier Management System is designed to automate and manage courier operations digitally. It eliminates manual record-keeping and provides a centralized platform to:

- Track shipments in real time
- Manage delivery agents
- Update parcel status
- Monitor delivery progress
- Maintain customer and shipment records

The system is built using Django’s MVC architecture, ensuring scalability, modularity, and maintainability.

## 🔄 System Workflow

1. Admin creates a shipment entry.
2. System generates a unique tracking ID.
3. Shipment is assigned to a delivery agent.
4. Status updates are recorded (Dispatched, In Transit, Delivered).
5. Customers can track shipment using tracking ID.
6. Delivery confirmation is logged in the system.
